## [Functions in MATLAB]
- See also [Function Basics]

```matlab
function [one,two,three] = myFunction(x)
    % Do some operations and modify one, two, three
end
```
- `[one, two, tree]` are the output variables
- `myFunction` is the function name
- `x` is the input arguments

## [Formatting Text]
### Precision
```matlab
var = 503.147;
printf('%.2f', var)
```

Output: `503.15`
